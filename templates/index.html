This API can normalize dates:

  • Normalize dates with different formats.
  • Converts dates to unix timestamp.
  • Converts unix timestamp to date.

API Usage Guide: /apidocs

Examples:


Convert date to unix timestamp:
URL/normalize/December 24 2019
Output:
{ "day": "2019-12-24", "input": "December 24 2019", "timestamp": 1577145600.0 }

Convert unix timestamp to date:
URL/normalize/1577145600
Output:
{ "day": "2019-12-24", "timestamp": "1577145600" }
